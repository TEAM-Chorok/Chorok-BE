Chorok은 식물 집사들을 위한 일정관리 및 커뮤니티 서비스입니다.

식물키우기가 처음이시거나 어려우셨던 분들도,

내 공간을 식물과 함께 더 예쁘게 꾸미고 싶은 분들도,

모두 초록과 함께 쉽고 편하게 식물을 가꿔보세요 :) !
